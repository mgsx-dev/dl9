# Game Hotkeys

F11 or ALT+ENTER - toggle fullscreen / windowed mode
ESCAPE - quit game

# Debug Hotkeys (DEV MODE only)

S - enable/disable Stats
Z - GL zzz silent errors
E - GL log Errors
R - GL Raise errors

N - next phase
B - restart game

P - pause

K - insta depth
W - insta win

Q - edit ingame settings (and pause)
